# UTools-Qwerty-Learner 项目详细策划案

## 一、项目概述

### 1.1 项目定位

**UTools-Qwerty-Learner** 是一款基于 UTools 平台的英语单词学习工具，将**打字练习**与**间隔重复记忆算法**相结合，帮助用户在练习打字的同时高效记忆英语单词。

### 1.2 核心价值

| 价值点 | 说明 |
|--------|------|
| 🎯 **打字+记忆双效合一** | 通过打字输入强化单词记忆，肌肉记忆+大脑记忆双重效果 |
| 📚 **科学复习机制** | 基于艾宾浩斯遗忘曲线的间隔重复算法，科学安排复习时间 |
| 🔄 **复习驱动新词** | 复习越多，可学新词越多，防止复习债堆积 |
| 📊 **学习数据可视化** | 完整的学习进度追踪和数据分析 |
| 📖 **词典集成** | 支持 MDX 词典查询，获取详细释义和音标 |

### 1.3 技术栈

```
┌─────────────────────────────────────────────────────────┐
│  前端框架: React 18 + TypeScript                         │
│  构建工具: Vite                                          │
│  状态管理: Jotai (原子化) + useImmerReducer              │
│  本地数据库: Dexie (IndexedDB 封装)                      │
│  样式方案: TailwindCSS                                   │
│  数据请求: SWR                                           │
│  平台集成: UTools API                                    │
└─────────────────────────────────────────────────────────┘
```

### 2.2 模块详解

#### 模块一：词库管理 (Gallery)

| 功能 | 说明 |
|------|------|
| 导入词库 | 支持文本文件导入，自动解析单词列表 |
| 词库列表 | 展示所有自定义词库及学习进度 |
| 词库删除 | 删除不需要的词库 |
| 进度统计 | 显示已学/已掌握/待复习数量 |

#### 模块二：打字学习 (Typing)

| 功能 | 说明 |
|------|------|
| 智能学习模式 | 自动判断复习/新词/巩固模式 |
| 打字练习 | 输入单词进行打字练习 |
| 发音播放 | 自动播放单词发音 |
| 进度追踪 | 实时更新学习进度 |
| 掌握标记 | 一键标记单词为已掌握（不占用学习名额） |

#### 模块三：词典查询 (MdxQuery)

| 功能 | 说明 |
|------|------|
| MDX 词典支持 | 加载本地 MDX 词典文件 |
| 单词查询 | 查询单词详细释义 |
| 多词典支持 | 同时使用多个词典 |

#### 模块四：数据分析 (Analysis)

| 功能 | 说明 |
|------|------|
| 学习统计 | 今日/本周/本月学习数据 |
| 热力图 | 学习活跃度可视化 |
| 趋势图 | 学习进度趋势分析 |

### 3.4 已掌握词不占名额机制

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    已掌握词不占名额                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  触发方式：用户在背单词界面点击"已掌握"按钮                                  │
│                                                                             │
│  规则：                                                                      │
│  - 复习时点击"已掌握"：该词不占用名额                                        │
│  - 学习新词时点击"已掌握"：自动补充新词                                      │
│  - 最终学习N个"未掌握"的词                                                   │
│                                                                             │
│  效果：                                                                      │
│  - 鼓励用户主动标记已熟悉的单词                                              │
│  - 确保用户学习的是真正需要学习的词                                          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3.5 固定上限型配置

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    固定上限型核心规则                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  1. 每日学习上限：默认20个词（可配置）                                        │
```

### 5.2 SM-2 算法改编

本项目的掌握度更新算法基于经典的 **SuperMemo SM-2** 算法改编：

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         SM-2 算法改编                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  输入参数:                                                                   │
│  - currentLevel: 当前掌握等级 (0-7)                                         │
│  - isCorrect: 本次是否正确                                                  │
│  - wrongCount: 本次错误次数                                                 │
│  - easeFactor: 当前难度因子 (1.3-3.0)                                       │
│                                                                             │
│  算法逻辑:                                                                   │
│                                                                             │
│  if (isCorrect && wrongCount === 0) {                                      │
│    // 完全正确: 升级 + 增加难度因子                                          │
│    newLevel = min(currentLevel + 1, 7)                                     │
│    newEaseFactor = min(easeFactor + 0.1, 3.0)                              │
│  } else if (isCorrect && wrongCount > 0) {                                 │
│    // 有错误但最终正确: 保持等级 + 降低难度因子                              │
│    newLevel = currentLevel                                                 │
│    newEaseFactor = max(easeFactor - 0.1, 1.3)                              │
│  } else {                                                                   │
│    // 错误: 降级 + 降低难度因子                                              │
│    newLevel = max(currentLevel - 1, 0)                                     │
│    newEaseFactor = max(easeFactor - 0.2, 1.3)                              │
│  }                                                                          │
│                                                                             │
│  下次复习时间:                                                               │
│  nextReviewTime = now + REVIEW_INTERVALS[newLevel] * easeFactor * 24h      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 八、总结

本项目实现了一个完整的**打字+背单词**学习系统，核心特点：

1. **科学的学习算法**：基于艾宾浩斯遗忘曲线和 SM-2 算法，智能安排复习时间
2. **复习驱动新词**：通过配额机制防止复习债堆积，保证学习效率
3. **完整的数据追踪**：记录每个单词的学习进度，支持数据分析
4. **良好的用户体验**：打字练习强化记忆，发音辅助学习
