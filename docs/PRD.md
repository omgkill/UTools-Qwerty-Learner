## 一、产品定位

### 1.1 产品概述

UTools-Qwerty-Learner 是一款基于 uTools 平台的单词学习工具，通过打字练习帮助用户记忆英语单词。用户在输入单词的过程中建立肌肉记忆，同时通过间隔重复算法巩固词汇。

### 1.2 目标用户

| 用户画像 | 描述 | 核心诉求 |
|---------|------|---------|
| 键盘工作者 | 程序员、文字工作者 | 边练打字边学英语，一举两得 |
| 英语学习者 | 备考四六级、考研、托福、GRE | 高效记忆单词，科学复习 |
| 上班族 | 工作间隙学习 | 隐蔽学习（摸鱼模式） |
| 语言爱好者 | 学习日语、德语等 | 多语言词汇学习 |

### 1.3 核心价值

- **肌肉记忆**：通过打字强化单词记忆，比传统阅读更深刻
- **间隔重复**：科学的复习算法，提高长期记忆效果
- **碎片学习**：uTools 快速唤起，随时开始学习
- **沉浸体验**：摸鱼模式透明窗口，上班学习两不误

### 2.2 功能详细说明

#### 2.2.1 正常学习模式（P0）

**功能描述**：核心学习模式，基于间隔重复算法安排学习内容。

**学习逻辑**：

```
1. 优先获取「待复习单词」→ 到期需要复习的单词
2. 若配额剩余 → 学习「新单词」
3. 若无待复习/新单词 → 学习完成
```

**关键参数**：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 每日上限 | 20 词 | 每日新学 + 复习的总上限 |
| 掌握等级 | 8 级 | NEW → LEARNED → FAMILIAR → KNOWN → PROFICIENT → ADVANCED → EXPERT → MASTERED |
| 复习间隔 | 0/1/2/4/7/15/30 天 | 根据掌握等级自动安排下次复习时间 |

**状态显示**：

- 当前学习类型（复习/新词）
- 今日学习数量（新学 + 复习）
- 已掌握数量
- 待复习数量

#### 2.2.2 重复学习模式（P1）

**功能描述**：重复练习今日学过的所有单词，巩固当日记忆。

**入口**：uTools 输入 `重复学习` 或 `repeat`

**数据来源**：今日学习记录（`DailyRecord.todayWords`）

**与正常学习的区别**：

| 对比项 | 正常学习 | 重复学习 |
|--------|---------|---------|
| 单词来源 | 到期单词 + 新单词 | 今日学过的单词 |
| 记录影响 | 更新进度、统计 | 不影响进度 |
| 适用场景 | 日常学习 | 当日巩固 |

#### 2.2.3 巩固学习模式（P1）

**功能描述**：复习已学过但未完全掌握的单词（掌握等级 1-6）。

**入口**：uTools 输入 `巩固学习` 或 `consolidate`

**数据来源**：所有学习进度（`getAllProgress`）

**筛选条件**：
- 掌握等级 > 0（已学过）
- 掌握等级 < 7（未完全掌握）

#### 2.2.4 摸鱼模式（P1）

**功能描述**：创建透明悬浮窗口，可在工作时隐蔽学习。

**入口**：uTools 输入 `moyu` / `moyv` / `typing-摸鱼模式`

**技术实现**：

```
透明窗口 + 始终置顶 + 无边框
自动开启沉浸模式
```

**配合功能**：
- 窗口透明度可调（需要原生模块支持）
- `Esc` 退出摸鱼窗口

#### 2.2.5 词典系统

**内置词典**：

| 分类 | 词典示例 |
|------|---------|
| 中国考试 | CET-4, CET-6, 考研英语, BEC |
| 出国考试 | TOEFL, GRE, IELTS |
| 日语 | N1, N2, N3, N4, N5 |
| 德语 | A1, A2, B1, B2, C1 |
| 编程词汇 | JavaScript, Python, Go, Rust |

**自定义词典**：

- 支持格式：JSON、Excel (.xlsx)
- 必填字段：`name`（单词）、`trans`（翻译）
- 可选字段：`usphone`（美式音标）、`ukphone`（英式音标）

**MDX 词典**：

- 支持导入 .mdx 格式词典
- 独立查询入口（`查词典`）
- 词典管理入口（`mdx管理`）

#### 2.2.6 错题记录

**触发机制**：用户打字错误时自动记录

**存储结构**：

```typescript
interface ErrorWord {
  id?: number
  dictId: string    // 所属词典
  word: IWord       // 单词详情
  count: number     // 错误次数
}
```

**复习入口**：
1. 章节结束后选择「重练错词」
2. 词典详情页点击「错题本」
3. 首页错题本词典入口

#### 2.2.7 学习统计

**热力图**：

| 颜色 | 含义 |
|------|------|
| 无色 | 当天无练习 |
| 浅绿 | 练习 1-2 次 |
| 中绿 | 练习 3-5 次 |
| 深绿 | 练习 6+ 次 |

**WPM 趋势**：

- 计算公式：`正确词数 / 用时(分钟)`
- 展示范围：近 30 天

**正确率**：

- 计算公式：`正确次数 / (正确次数 + 错误次数) × 100%`

## 三、用户流程

### 3.1 新用户首次使用流程

```
uTools 唤起 → 输入 "English" → 进入正常学习模式
                                    ↓
                            系统自动选择默认词典
                                    ↓
                            开始打字练习
                                    ↓
                            完成今日学习目标
```

### 3.2 日常学习流程

```
┌─────────────────────────────────────────────────────────────┐
│                     uTools 唤起插件                          │
│                          ↓                                   │
│              ┌────────────┴────────────┐                    │
│              ↓                         ↓                    │
│        正常学习模式                其他模式                   │
│     (输入 "English")          (repeat/consolidate/moyu)      │
│              ↓                         ↓                    │
│     ┌────────┴────────┐          进入对应模式               │
│     ↓                 ↓                                      │
│  有待复习词         无待复习词                                │
│     ↓                 ↓                                      │
│  复习到期单词      检查新词配额                               │
│     ↓                 ↓                                      │
│  配额剩余→学新词   配额充足→学新词                            │
│     ↓                 ↓                                      │
│  配额用尽→结束     无新词/配额用尽→结束                        │
└─────────────────────────────────────────────────────────────┘
```

### 3.3 打字练习交互流程

```
显示单词 + 音标 + 翻译
        ↓
用户逐字母输入
        ↓
┌───────┴───────┐
↓               ↓
正确输入       错误输入
↓               ↓
字母变绿      字母变红 + 清空当前输入
↓               ↓
下一字母      重新开始该词
↓               ↓
└───────┬───────┘
        ↓
单词完成 → 自动跳下一个词
        ↓
全部完成 → 显示结果页
```

### 3.4 快捷键汇总

| 快捷键 | 功能 | 适用场景 |
|--------|------|---------|
| `Tab` | 跳过当前词 | 打字练习中 |
| `Enter` | 开始/重复章节 | 等待开始时 |
| `Esc` | 退出练习/摸鱼窗口 | 任意时刻 |
| `Ctrl+1` | 查看详情 | 可配置 |
| `Ctrl+2` | 返回 | 可配置 |


#### 每日记录 (DailyRecord)

```typescript
interface DailyRecord {
  dict: string           // 词典 ID

  learnedCount: number   // 新学数量
  reviewedCount: number  // 复习数量
```

| 等级 | 名称 | 复习间隔 |

| 3 | 认识 | 4 天 |
| 4 | 熟练 | 7 天 |
| 5 | 精通 | 15 天 |
| 6 | 专家 | 30 天 |
| 7 | 已掌握 | 不再复习 |
