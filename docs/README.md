# UTools-Qwerty-Learner

UTools 平台打字背单词插件，通过键盘肌肉记忆强化单词学习。

## 功能列表

| 功能 | 说明 |
|------|------|
| 打字练习 | 选词典→选章节→逐词打字练习 |
| 多词典支持 | 预置 CET4/6、GRE、TOEFL 等 20+ 词典，支持自定义导入 |
| 错题记录 | 自动记录打错单词，支持专项复习 |
| 摸鱼模式 | 透明悬浮窗口，隐蔽学习 |
| 沉浸模式 | 隐藏无关 UI，专注练习 |
| 统计分析 | 热力图、WPM、正确率趋势图 |

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```
