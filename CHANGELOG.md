## [0.1.0] - 2026-03-08

### 新增

#### 学习功能
- **打字练习**：通过输入单词进行记忆训练
- **智能学习模式**：
  - 新词学习：学习未掌握的单词
  - 复习模式：基于间隔重复算法复习到期单词
  - 巩固模式：巩固已学但未完全掌握的单词
  - 重复学习：重复学习今日所学单词
- **每日目标**：每日学习上限 20 词
- **学习进度追踪**：记录每个单词的掌握等级、复习次数

#### 词库管理
- **多词库支持**：支持多种词库导入
- **自定义词库**：支持 JSON 和 Excel 格式导入
- **MDX 词典**：支持导入 MDX 格式的词典文件

#### 学习统计
- **三层统计视图**：
  1. 词库列表：显示所有学习过的词库
  2. 天数列表：显示选定词库的学习日期
  3. 单词详情：显示当天学习的单词
- **新词/复习区分**：自动区分新学单词和复习单词

#### 单词查询
- **MDX 查词**：支持 MDX 格式词典查词
- **详细释义**：在练习界面查看完整释义

### 特色功能

- **摸鱼模式**：自定义窗口透明度，适合工作间隙学习
- **沉浸模式**：隐藏无关组件，专注于单词记忆
- **发音功能**：支持美式/英式发音切换

### 技术实现

#### 状态管理
- 使用 Jotai 进行状态管理
- 状态分层架构：
  - 全局状态（词库配置、发音类型等）
  - 会话状态（单词列表、学习进度等）
  - UI 状态（组件内部状态）
- 单一数据源原则

#### 数据存储
- **uTools DB**：存储词库配置、MDX 词典
- **IndexedDB**：存储学习记录、单词进度
- 数据持久化，重启不丢失

#### 学习算法
- **间隔重复算法**：8 个掌握等级
- **复习间隔**：0小时、1天、2天、4天、7天、15天、30天
- **每日学习流程**：复习优先 → 新词学习 → 目标达成

### 测试

- 单元测试：核心逻辑函数测试
- 集成测试：组件协作测试
- 学习模拟测试：15天学习流程模拟
