## 2026-02-14 修复词典加载和 React Router 警告

**文件**: `src/pages/Typing/hooks/useWordList.ts`

**问题**: 自定义词典的 `languageCategory` 字段可能缺失，导致词典被错误地判断为远程词典，尝试从 `/dicts/` 路径加载。

**修改内容**:
```typescript
// 修改前
const isLocalDict = currentDictInfo ? currentDictInfo.languageCategory === 'custom' : false

// 修改后
const isLocalDict = currentDictInfo ? (currentDictInfo.id.startsWith('x-dict-') || currentDictInfo.languageCategory === 'custom') : false
```

**修复结果**: 自定义词典（ID 以 `x-dict-` 开头）会被正确识别为本地词典，从 uTools 数据库加载。

## 2026-02-14 导入词典逻辑说明

### 功能概述
用户可以通过拖拽或选择 Excel 文件导入自定义词典。

### 导入流程

```
用户选择 Excel 文件
         ↓
┌─────────────────────────────────────┐
│  1. 文件验证                          │
│  - 检查文件类型 (xls/xlsx/json)       │
│  - 检查文件大小 (≤2MB)                │
└─────────────────────────────────────┘
         ↓
┌─────────────────────────────────────┐
│  2. Excel 转 JSON                     │
│  - 使用 XLSX 库解析 Excel             │
│  - 验证表头必须有 name 和 trans 列    │
│  - 转换为标准 JSON 格式               │
└─────────────────────────────────────┘
         ↓
┌─────────────────────────────────────┐
│  3. 创建词典元数据                    │
│  - 生成唯一 ID (x-dict-{uuid})       │
│  - 计算章节数量 (每章20词)            │
└─────────────────────────────────────┘
         ↓
┌─────────────────────────────────────┐
│  4. 存储到 uTools 数据库              │
│  - 词典内容: utools.db.put({_id, data})│
│  - 配置列表: local-dict-config        │
└─────────────────────────────────────┘
```

### 涉及文件

| 文件 | 功能 |
|------|------|
| `Form4AddDict/index.tsx` | 表单界面、文件选择、验证 |
| `convertExcelToJson.ts` | Excel 转 JSON 格式 |
| `preload.js` | uTools 数据库存储操作 |

### Excel 格式要求

| 列名 | 必填 | 说明 |
|------|------|------|
| name | ✅ | 单词 |
| trans | ✅ | 翻译 |
| usphone | ❌ | 美式音标 |
| ukphone | ❌ | 英式音标 |
| notation | ❌ | 注解 |

### JSON 输出格式

```json
[
  {
    "name": "apple",
    "trans": ["苹果"],
    "usphone": "'æpl",
    "ukphone": "'æpl"
  }
]
```

### 存储结构

```
uTools 数据库
├── local-dict-config     # 词典配置列表
│   └── [{id, name, length, chapterCount, ...}, ...]
│
├── x-dict-{uuid-1}       # 词典1内容
│   └── [{name, trans, usphone, ukphone}, ...]
│
└── x-dict-{uuid-2}       # 词典2内容
    └── [{name, trans, usphone, ukphone}, ...]
```

### 关键函数

#### `convertExcelToJson(file: File)`
- 使用 XLSX 库读取 Excel 文件
- 验证表头格式
- 转换为 JSON 数组

#### `saveDict(formData, jsonData)`
- 创建词典元数据 (id, name, length, chapterCount 等)
- 调用 `window.newLocalDictFromJson` 保存

#### `window.newLocalDictFromJson(jsonData, dictMeta)`
- 保存词典内容到 uTools 数据库
- 更新词典配置列表
- 检查是否已存在，避免重复添加

## 2026-02-14 词典管理添加移动动画

### 修改目的
为词典管理页面的词典上下移动添加 FLIP 动画效果，提升用户体验。

### 修改内容

**文件**: `src/pages/MdxManage/index.tsx`

**修改方案**:
使用 FLIP (First, Last, Invert, Play) 动画技术实现列表项位置过渡动画：
1. **First**: 移动前记录所有元素的位置 (`recordPositions`)
2. **Last**: 数据更新后获取元素新位置
3. **Invert**: 计算位置差值，用 `transform: translateY()` 反向偏移
4. **Play**: 移除偏移，让 CSS transition 播放动画

**代码实现**:
```typescript
// 记录元素位置
const recordPositions = useCallback(() => {
  const items = listRef.current.querySelectorAll('.dict-item')
  items.forEach((item) => {
    const path = item.getAttribute('data-path')
    if (path) {
      prevPositions.current.set(path, item.getBoundingClientRect())
    }
  })
}, [])

// 播放 FLIP 动画
const playFlipAnimation = useCallback(() => {
  const items = listRef.current.querySelectorAll('.dict-item')
  items.forEach((item) => {
    const prevRect = prevPositions.current.get(path)
    if (!prevRect) return
    
    const currentRect = item.getBoundingClientRect()
    const deltaY = prevRect.top - currentRect.top
    
    if (deltaY !== 0) {
      item.style.transform = `translateY(${deltaY}px)`
      item.style.transition = 'none'
      
      requestAnimationFrame(() => {
        item.style.transition = 'transform 0.3s ease'
        item.style.transform = 'translateY(0)'
      })
    }
  })
}, [])
```

### 修改结果
- 点击上移/下移按钮时，词典项平滑过渡到新位置
- 动画时长 0.3s，使用 ease 缓动函数

## 2026-02-14 修复查词典页面无法获取查询词

### 修改目的
修复查词典页面显示"未查到结果"的问题，原因是无法获取 uTools 传入的查询词。

### 问题分析
1. voca-plugin 的 `onPluginEnter` 回调传递完整的 action 对象 `{ code, payload }`
2. 当前项目只传递了 code，没有传递 payload
3. 导致查词典页面无法获取要查询的单词

### 修改内容

#### 1. 更新 preload.js
**文件**: `public/preload.js`

传递完整的 action 对象：
```javascript
let currentAction = null;
utools.onPluginEnter((action) => {
  currentMode = action.code;
  currentAction = action;
  window.dispatchEvent(new CustomEvent('utools-mode-change', { detail: action }));
});
window.getAction = () => currentAction;
```

#### 2. 更新 MdxQueryPage
**文件**: `src/pages/MdxQuery/index.tsx`

在组件初始化时获取 action：
```typescript
useEffect(() => {
  // 监听模式切换
  const handleModeChange = (e: CustomEvent) => {
    const action = e.detail
    if (action?.payload) {
      handleSearch(String(action.payload).trim())
    }
  }
  window.addEventListener('utools-mode-change', handleModeChange)
  
  // 初始化时获取当前 action
  const action = window.getAction?.()
  if (action?.payload) {
    handleSearch(String(action.payload).trim())
  }
  
  return () => {
    window.removeEventListener('utools-mode-change', handleModeChange)
  }
}, [handleSearch])
```

### 修改结果
- 在 uTools 中选中单词后选择"查词典"，能正确获取查询词
- 查询结果正常显示
