┌─────────────────────────────────────────────────────────────┐
│  第1天学习流程详解（固定上限型配置）                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  配置参数：                                                  │
│  - 每日学习上限：20个词（可在设置中配置）                     │
│                                                             │
│  初始状态：                                                  │
│  - 到期词：0个（词库为空）                                   │
│  - 新词配额：20 - 0 = 20个                                   │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  学习过程：学习单词1-20（20个新词）                          │
│                                                             │
│  今日统计：                                                  │
│  - 新词：20个                                                │
│  - 复习：0个                                                 │
│  - 总计：20个                                                │
│                                                             │
│  单词状态：单词1-20等级1，下次复习：1小时后                  │
│                                                             │
│  用户可随时退出，进度自动保存                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  固定上限型核心规则                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 每日学习上限：默认20个词（可配置）                        │
│  2. 复习优先：有到期词先复习                                 │
│  3. 新词配额 = max(0, 上限 - 当日已复习数 - 当日已学习新词数) │
│  4. 额外复习不计入总量，不影响新词配额                        │
│  5. 已掌握词不计入当日复习计划                                │
│  6. 无硬性目标：用户想学多少就学多少                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  已掌握词不计入当日复习计划                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  定义：背单词过程中点击"已掌握"的单词                        │
│                                                             │
│  触发方式：                                                  │
│  - 用户在背单词界面点击"已掌握"按钮                          │
│  - 该单词立即标记为已掌握状态                                │
│                                                             │
│  规则：                                                      │
│  - 点击"已掌握"：该词不计入当日复习计划                      │
│  - 学习新词时点击"已掌握"：自动补充新词                      │
│  - 最终学习N个"未掌握"的词                                   │
│                                                             │
│  目的：                                                      │
│  - 鼓励用户主动标记已熟悉的单词                              │
│  - 确保用户学习的是真正需要学习的词                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  额外复习机制                                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  触发条件：到期词 > 每日上限                                  │
│                                                             │
│  流程：                                                      │
│  1. 复习前N个到期词（计入总量，N=每日上限）                   │
│  2. 提示："您还有X个单词需要复习，是否继续？"                 │
│  3. 用户选择：                                               │
│     - 继续复习：额外复习不计入总量                           │
│     - 稍后再说：明天继续复习                                 │
│                                                             │
│  效果：可利用空闲时间快速消化堆积                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
