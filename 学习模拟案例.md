# 100个单词学习模拟案例（固定上限型配置）

## 配置参数

| 参数 | 值 | 说明 |
|------|-----|------|
| DAILY_LIMIT | 20 | 每日学习上限（可配置） |

## 核心规则

```
┌─────────────────────────────────────────────────────────────┐
│  固定上限型核心规则                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 每日学习上限：默认20个词（可配置）                        │
│  2. 复习优先：有到期词先复习                                 │
│  3. 新词配额 = max(0, 上限 - 当日已复习数)                   │
│  4. 额外复习不计入总量，不影响新词配额                        │
│  5. 已掌握词不占名额：                                       │
│     - 用户点击"已掌握"按钮的词不占用名额                     │
│     - 学习新词时点击"已掌握"，自动补充新词                   │
│     - 最终学习N个"未掌握"的词                                │
│  6. 无硬性目标：用户想学多少就学多少                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 复习间隔规则

| 等级 | 间隔 | 状态 |
|-----|------|------|
| 0 | 立即 | 新词 |
| 1 | 1小时 | 初学 |
| 2 | 1天 | 熟悉 |
| 3 | 2天 | 认识 |
| 4 | 4天 | 熟练 |
| 5 | 7天 | 精通 |
| 6 | 15天 | 专家 |
| 7 | 30天 | 已掌握 |

## 额外复习场景演示

假设用户连续7天未登录，有35个到期词：

```
┌─────────────────────────────────────────────────────────────┐
│  额外复习场景                                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  到期词：35个                                                │
│  新词配额：20 - 0 = 20个（初始）                             │
│                                                             │
│  学习过程：                                                  │
│                                                             │
│  1. 复习前20个到期词（计入总量）                             │
│     - 当日已复习：20个                                       │
│     - 新词配额：20 - 20 = 0                                 │
│     - 剩余到期词：15个                                       │
│                                                             │
│  2. 系统提示：                                               │
│     "您还有15个单词需要复习，是否继续？"                     │
│     [继续复习] [稍后再说]                                    │
│                                                             │
│  3a. 用户选择"继续复习"：                                    │
│     - 额外复习15个词（不计入总量）                           │
│     - 当日统计：复习35个（20计入+15额外）                    │
│     - 新词配额：仍为0                                        │
│     - 到期词清零，明天新词配额恢复                           │
│                                                             │
│  3b. 用户选择"稍后再说"：                                    │
│     - 当日统计：复习20个                                     │
│     - 剩余到期词：15个（明天继续）                           │
│     - 明天新词配额：20 - 15 = 5个                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
