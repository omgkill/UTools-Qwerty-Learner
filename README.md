# UTools-Qwerty-Learner

一款基于 uTools 平台的单词学习工具，通过打字练习帮助用户记忆英语单词。

## 功能特性

### 核心功能

#### 📚 单词学习
- **打字练习**：通过输入单词进行记忆训练
- **智能学习模式**：
  - 新词学习：学习未掌握的单词
  - 复习模式：复习到期单词（基于间隔重复算法）
  - 巩固模式：巩固已学但未完全掌握的单词
- **每日目标**：每日学习上限 20 词
- **学习进度追踪**：记录每个单词的掌握等级、复习次数、错误次数

#### 📖 词库管理
- **内置词库**：支持多种词库导入
- **自定义词库**：支持 JSON 和 Excel 格式导入
- **MDX 词典**：支持导入 MDX 格式的词典文件
- **错题本**：自动记录错题，支持章节错题同步

#### 📊 学习统计
- **三层统计视图**：
  1. 词库列表：显示所有学习过的词库
  2. 天数列表：显示选定词库的学习日期
  3. 单词详情：显示当天学习的单词
- **新词/复习区分**：自动区分新学单词和复习单词
- **学习进度可视化**：显示掌握等级、错误次数等

#### 🔍 单词查询
- **MDX 查词**：支持 MDX 格式词典查词
- **详细释义**：在练习界面点击"详细"查看完整释义
- **快捷键支持**：ESC 返回

### 特色功能

#### 🎭 摸鱼模式
- 提供额外的入口，支持自定义窗口透明度
- 适合在工作间隙学习

#### 🌙 沉浸模式
- 隐藏练习过程中无关的组件
- 专注于单词记忆

#### 🔊 发音功能
- 支持美式/英式发音切换
- 支持自动播放发音

## 项目结构

```
src/
├── components/          # 公共组件
├── pages/
│   ├── Typing/          # 打字练习页面
│   ├── Analysis/        # 学习统计页面
│   ├── MdxQuery/        # MDX 查词页面
│   └── MdxManage/       # MDX 词典管理页面
├── hooks/               # 自定义 Hooks
├── store/               # 状态管理 (Jotai)
├── utils/
│   ├── db/              # IndexedDB 数据库操作
│   └── ...              # 工具函数
├── dict/                # 词典适配器
└── services/            # 业务服务
```

## 技术栈

- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **状态管理**：Jotai + useImmerReducer
- **数据库**：IndexedDB (Dexie.js)
- **样式**：TailwindCSS
- **测试**：Vitest + Testing Library

## 部署方式

### uTools 插件部署

1. **构建项目**
   ```bash
   npm run build
   ```

2. **打包插件**
   ```bash
   npm run utools
   ```
   此命令会将 `build` 目录打包为 uTools 插件格式。

3. **安装插件**
   - 将生成的 `.upx` 文件拖入 uTools 窗口
   - 或者在 uTools 开发者工具中选择 `plugin.json` 所在目录

### 开发调试

在 uTools 开发者工具中：
1. 选择项目目录
2. 点击"运行"启动插件
3. 修改代码后自动热更新

## 学习算法

### 间隔重复算法

单词掌握等级分为 8 级：

| 等级 | 名称 | 复习间隔 |
|------|------|----------|
| 0 | NEW | - |
| 1 | LEARNED | 1小时 |
| 2 | FAMILIAR | 1天 |
| 3 | KNOWN | 2天 |
| 4 | PROFICIENT | 4天 |
| 5 | ADVANCED | 7天 |
| 6 | EXPERT | 15天 |
| 7 | MASTERED | 30天 |

### 每日学习流程

1. **复习优先**：优先复习到期的单词
2. **新词学习**：复习完成后学习新词
3. **目标达成**：每日最多学习 20 个单词（新词+复习）

### 数据持久化

- **IndexedDB**：存储学习记录、单词进度、每日统计
- **uTools 数据库**：存储词库配置、MDX 词典
- 数据在重启电脑后不会丢失

## 故障排除

### 插件闪退

1. 检查 `plugin.json` 配置是否正确
2. 确认 `preload.js` 路径正确
3. 查看控制台错误日志

### 数据丢失

- 学习数据存储在 IndexedDB，不会因插件更新丢失
- 词库配置存储在 uTools 数据库，卸载插件后仍保留

### MDX 词典无法加载

1. 确认词典文件格式正确
2. 检查词典路径配置
3. 在 MDX 管理页面重新添加
