# UTools-Qwerty-Learner

本插件(UTools-Qwerty-Learner)为UTools平台下专用插件，基于Qwerty-Learner项目修改移植并进行二次开发而来。

主要新增功能：

1. 摸鱼模式，为用户提供额外的进入入口，此模式下程序可自定义整体透明。
2. 沉浸模式，开启此模式后，不显示练习过程中无关的组件。
3. 词典错题记录，可为每个词典开启错题的记录，并在章节练习结束后自动同步章节的错词。
4. 自定义词典导入（支持Json和Excel表格），程序中提供模板方便用户进行修改导入。

## 故障修复

### 2026-02-13 修复插件无法打开问题

**问题**：uTools 插件无法打开，提示缺少预加载脚本。

**原因**：`build` 目录中缺少 `preload.js` 文件，而 `plugin.json` 配置文件中明确指定了此文件作为预加载脚本。

**解决方案**：将 `public/preload/services.js` 的内容复制到 `build/preload.js` 文件中，确保插件能够正常加载所需的 Node.js 能力（如文件系统操作）。

**验证**：插件现在可以正常在 uTools 中打开和使用，所有功能均可正常访问。
