# UTools-Qwerty-Learner

UTools 平台打字背单词插件，通过键盘肌肉记忆强化单词学习。

## 文档

完整文档位于 [docs/](./docs/) 目录：

- [项目概述](./docs/README.md)
- [业务逻辑](./docs/business.md) - 开发测试必读
- [技术架构](./docs/architecture.md)

## 快速开始

```bash
npm install
npm run dev
```

## 功能

| 功能 | 说明 |
|------|------|
| 打字练习 | 选词典→选章节→逐词打字 |
| 多词典支持 | CET4/6、GRE、TOEFL 等 170+ 词典 |
| 错题记录 | 自动记录打错单词 |
| 摸鱼模式 | 透明悬浮窗口 |
| 统计分析 | 热力图、WPM 趋势 |
