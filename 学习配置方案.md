# 学习配置方案

## 一、推荐方案：固定上限型

### 核心规则

```
┌─────────────────────────────────────────────────────────────┐
│  固定上限型核心规则                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 每日学习上限：默认20个词（可在设置中配置）                │
│                                                             │
│  2. 复习优先：有到期词先复习                                 │
│                                                             │
│  3. 新词配额 = max(0, 每日上限 - 当日已复习数 - 当日已学习新词数)│
│     - 当日已复习数只计入前N个（N=每日上限）                  │
│     - 额外复习不计入                                         │
│                                                             │
│  4. 额外复习机制：                                           │
│     - 当到期词 > 每日上限时，复习完上限数量后                │
│     - 用户可自愿继续复习剩余到期词                           │
│     - 额外复习不计入总量，不影响新词配额                     │
│                                                             │
│  5. 已掌握词不计入当日复习计划：                             │
│     - 背单词过程中点击"已掌握"即视为已掌握                 │
│     - 该词不计入当日复习计划                                 │
│     - 不消耗学习配额                                         │
│     - 鼓励用户保持对已掌握单词的复习                         │
│                                                             │
│  6. 无硬性目标：用户想学多少就学多少                         │
│                                                             │
│  7. 巩固模式（可选）：                                       │
│     - 当无到期词且无新词时，可进入巩固模式                   │
│     - 也可直接结束当天学习                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 配置参数

```typescript
export const LEARNING_CONFIG = {
  DAILY_LIMIT: 20,              // 每日学习上限（可配置）
}
```

```
┌─────────────────────────────────────────────────────────────┐
│  固定上限型学习流程                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  打开应用                                                    │
│      │                                                      │
│      ▼                                                      │
│  检查到期词数量                                              │
│      │                                                      │
│      ▼                                                      │
│  计算新词配额                                                │
│      │                                                      │
│      │  新词配额 = max(0, DAILY_LIMIT - 当日已复习数 - 当日已学习新词数) │
│      │  当日已复习数只计入前DAILY_LIMIT个                    │
│      │  背单词中点击"已掌握"不计入当日复习计划               │
│      │                                                      │
│      ▼                                                      │
│  有到期词？                                                  │
│      │                                                      │
│      ├─ 是 → 复习到期词（优先）                              │
│      │      │                                               │
│      │      ├─ 复习数 < DAILY_LIMIT                         │
│      │      │      │                                        │
│      │      │      └─ 继续复习，消耗配额                    │
│      │      │      （点击"已掌握"不计入当日复习计划）        │
│      │      │                                               │
│      │      └─ 复习数 = DAILY_LIMIT 且 还有到期词            │
│      │             │                                        │
│      │             └─ 提示：是否继续额外复习？              │
│      │                    （额外复习不计入总量）             │
│      │                                                      │
│      └─ 否 → 有新词且新词配额>0？                            │
│             │                                                │
│             ├─ 是 → 学习新词（最多 = 新词配额）              │
│             │                                                │
│             └─ 否 → 巩固模式（可选） / 结束当天学习           │
│                                                             │
│  用户可随时退出，进度自动保存                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 三、已掌握词不计入当日复习计划机制

### 规则说明

```
┌─────────────────────────────────────────────────────────────┐
│  已掌握词不计入当日复习计划                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  定义：背单词过程中点击"已掌握"的单词                        │
│                                                             │
│  触发方式：                                                  │
│  - 用户在背单词界面点击"已掌握"按钮                          │
│  - 该单词立即标记为已掌握状态                                │
│                                                             │
│  规则：                                                      │
│  - 点击"已掌握"：该词不计入当日复习计划                      │
│  - 学习新词时点击"已掌握"：自动补充新词                      │
│  - 最终学习N个"未掌握"的词                                   │
│                                                             │
│  目的：                                                      │
│  - 鼓励用户主动标记已熟悉的单词                              │
│  - 确保用户学习的是真正需要学习的词                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 场景演示

```
┌─────────────────────────────────────────────────────────────┐
│  已掌握词不计入当日复习计划场景                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  假设：                                                      │
│  - 每日上限：20个                                            │
│  - 到期词：15个                                              │
│                                                             │
│  学习过程：                                                  │
│                                                             │
│  1. 复习15个到期词                                           │
│     - 复习过程中，用户点击5个"已掌握"按钮                    │
│     - 这5个已掌握词不计入当日复习计划                        │
│     - 实际消耗名额：15 - 5 = 10个                            │
│     - 新词配额：20 - 10 = 10个                               │
│                                                             │
│  2. 学习10个新词                                             │
│     - 学习过程中，用户点击3个"已掌握"按钮                    │
│     - 这3个已掌握词不计入当日复习计划                        │
│     - 系统自动补充3个新词                                    │
│     - 继续学习...                                            │
│     - 如果补充的新词中又有"已掌握"，继续补充                 │
│                                                             │
│  3. 最终结果                                                 │
│     - 用户学习了10个"未掌握"的新词                           │
│     - 这10个新词都是真正需要学习的                            │
│                                                             │
│  效果：                                                      │
│  - 已掌握词不计入当日复习计划                                │
│  - 用户最终学习的是N个"未掌握"的词                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 四、额外复习机制详解

### 触发条件

```
到期词数量 > DAILY_LIMIT（默认20）
```

### 机制说明

```
┌─────────────────────────────────────────────────────────────┐
│  额外复习机制                                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  场景：用户有35个到期词                                      │
│                                                             │
│  第一步：复习前20个（计入总量）                              │
│    - 消耗20个学习配额                                        │
│    - 新词配额 = 20 - 20 = 0                                 │
│                                                             │
│  第二步：提示额外复习                                        │
│    - "您还有15个单词需要复习，是否继续？"                    │
│    - 用户可选择：继续复习 / 稍后再说                         │
│                                                             │
│  第三步：额外复习（不计入总量）                              │
│    - 复习剩余15个词                                          │
│    - 不消耗学习配额                                          │
│    - 不影响新词配额（仍为0）                                 │
│    - 但可以快速消化堆积                                      │
│                                                             │
│  效果：                                                      │
│    - 用户可以利用空闲时间快速消化堆积                        │
│    - 不会因为堆积而无法学习新词                              │
│    - 复习越多，到期词越少，新词配额恢复越快                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 五、详细模拟

### 第1天

```
到期词：0个
新词配额：20 - 0 = 20个

学习过程：学习单词1-20（20个新词）
今日统计：新词20个，复习0个，总计20个
单词状态：单词1-20等级1，下次复习：1天后
```

---

### 第2天

```
到期词：20个（单词1-20到期）
新词配额：20 - 20 = 0个

学习过程：复习单词1-20（等级1→2）
今日统计：新词0个，复习20个，总计20个
单词状态：单词1-20等级2，下次复习：第3天
```

---

### 第3天

```
到期词：0个（单词1-20等级2，1天后到期）
新词配额：20 - 0 = 20个

学习过程：学习单词21-40（20个新词）
今日统计：新词20个，复习0个，总计20个
累计已学：40个新词
```

## 九、方案优缺点

### 优点

| 优点 | 说明 |
|------|------|
| 规则简单 | 每天最多N个词，复习优先 |
| 学习速度较快 | 学完100个新词只需14天 |
| 复习压力可控 | 每天最多N个词，不会爆炸 |
| 无硬性目标 | 用户想学多少就学多少 |
| 额外复习机制 | 可利用空闲时间快速消化堆积 |
| 可配置 | 用户可根据自己情况调整上限 |

### 缺点

| 缺点 | 说明 |
|------|------|
| 新词学习呈脉冲式 | 可能连续多天只复习 |
